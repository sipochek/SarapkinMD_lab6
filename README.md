1.Вариант 29.
Написать программу, которая, используя метод бинарного поиска, выполняет
поиск в упорядоченном по возрастанию массиве. Массив предварительно
сортируется методом «пузырька».
--ТЕСТЫ:
1) arr[10] = {73 ,21, 45, 89, 12, 67, 34, 56, 90, 8} - ищем число 45.
Результат: Число 45 в позиции 4.

3) arr[10] = {15, 42, 7, 93, 28, 64, 51, 3, 86, 39} - ищем 64.
Результат: Число 64 в позиции 7

3) arr[10] = { -5, 3, -2, 8, -1, 0, 6, -3, 4, -4 }
Результат: число 0 в позиции 5.
